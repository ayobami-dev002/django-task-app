# 📝 Django Task Manager

A simple task manager web app built with Django.  
Helps you create, manage, and delete tasks efficiently.


## 🚀 Features

- ✅ Add new tasks
- 📋 View task list
- ❌ Delete tasks
- 📦 Built with Django
- 💄 Styled with Bootstrap


## 🧰 Tech Stack

- Python 3.x
- Django 4.x
- Bootstrap 5
